Velosurf 2.x

The latest release is Velosurf 2.2 (gzipped tar archive). It contains both the binary and the sources.

Velosurf 2.x need Java 1.5. If you're still using an older Java version, consider using a Velosurf 1.4.x release.

If you are upgrading from a 1.4.x release and using Velosurf inside a Velocity Tools webapp, be sure to upgrade your version of velocity-tools-view and to change the deprecated classname velosurf.tools.VelosurfTool to the new velosurf.web.VelosurfTool inside toolbox.xml.

The dependency jars are not included in the archive. To automatically download them, if you have Apache Ant installed, go into the build/ directory of the extracted archive and type ant download. This will download all dependency jars into the ./lib/ directory. You can also manually download dependencies (gzipped tar archive, meant to be extracted at the base directory of the release) or fetch them individually from the links bellow:

  • Dependencies needed to build velosurf:
    • commons-lang-2.2.jar
    • jdom-1.0.jar
    • servletapi-2.4.jar
    • velocity-1.4.jar
    • velocity-tools-view-1.3.jar has not yet been released at the time this page is being written. velocity-tools-view-1.3-dev.jar is provided in build/lib.
  • Dependencies needed at runtime:
    • commons-beanutils-1.7.0.jar (needed by Velocity Tools)
    • commons-collections-3.2.jar (needed by Velocity)
    • commons-digester-1.7.jar (needed by Velocity Tools)
    • crimson-1.1.3.jar (needed by JDOM)

Some earlier versions of the commons-* libraries may be supported by Velosurf. To test them, you can edit the version numbers in the ./build/build.properties file and test Velosurf with 'ant test'.

Running the tests also requires some additionnal libraries which will be downloaded when needed.

You can also checkout the last sources from the SourceForge subversion repository with the following command:

svn co https://velosurf.svn.sourceforge.net/svnroot/velosurf/trunk velosurf

Velosurf 1.4.x

The Velosurf 1.4.x releases are still maintained for bugs but not for functional upgrades.

The latest release is Velosurf 1.4.4. (gzipped tar archive). It contains both the binary and the sources, plus all dependency jars in the lib/ directory. The binary is in the bin/ directory of the extracted archive.

You can also checkout the last 1.4.x sources from the SourceForge CVS repository with the following commands:

cvs -d:pserver:anonymous@velosurf.cvs.sf.net:/cvsroot/velosurf login

cvs -z3 -d:pserver:anonymous@velosurf.cvs.sf.net:/cvsroot/velosurf co -P velosurf
