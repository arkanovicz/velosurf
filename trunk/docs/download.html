Velosurf 2.x

The latest release is Velosurf 2.0 (gzipped tar archive). It contains both the binary and the sources.

Velosurf 2.x need Java 1.5. If you're still using an older Java version, consider using a Velosurf 1.4.x release.

The dependency jars are not included in the archive. To automatically download them, if you have Apache Ant installed, go into the build/ directory of the extracted archive and type ant download. This will download all dependency jars into the lib/ directory. You can also manually download all dependencies (gzipped tar archive) or fetch them individually:

  • commons-beanutils-1.7.0.jar
  • commons-collections-3.2.jar
  • commons-lang-2.2.jar
  • commons-logging-1.1.jar
  • crimson-1.1.3.jar
  • jdom-1.0.jar
  • servletapi-2.4.jar
  • velocity-1.4.jar
  • velocity-tools-view-1.3.jar

You can also checkout the last sources from the SourceForge subversion repository with the following command:

svn co https://velosurf.svn.sourceforge.net/svnroot/velosurf/trunk velosurf

Velosurf 1.4.x

The Velosurf 1.4.x releases are still maintained for bugs but not for functional upgrades.

The latest release is Velosurf 1.4.4. (gzipped tar archive). It contains both the binary and the sources, plus all dependency jars in the lib/ directory. The binary is in the bin/ directory of the extracted archive.

You can also checkout the last 1.4.x sources from the SourceForge CVS repository with the following commands:

cvs -d:pserver:anonymous@velosurf.cvs.sf.net:/cvsroot/velosurf login

cvs -z3 -d:pserver:anonymous@velosurf.cvs.sf.net:/cvsroot/velosurf co -P velosurf
